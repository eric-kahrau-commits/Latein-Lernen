# Latein Lernen

Moderne TypeScript-Website mit responsiver Menüleiste.

## Starten

```bash
npm install
npm run dev
```

Im Browser: [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```

## Funktionen

- **Obere Menüleiste** mit Hamburger-Button (öffnet/schließt die Sidebar auf Mobilgeräten)
- **Sidebar** mit 5 Punkten: Home, Lernen, Neu, Statistiken, Einstellungen (jeweils mit Icon)
- **Desktop (ab 1024px):** Sidebar ist immer sichtbar, nur Icons; beim Hover erscheint die volle Breite mit Beschriftungen
- **Mobil:** Sidebar erscheint per Overlay beim Tipp auf den Hamburger
- Weiche Animationen für Sidebar, Hamburger und Hover-Zustände
